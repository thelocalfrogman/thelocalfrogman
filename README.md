## 🔥 Homelabs for Hackers

> My flagship open source project... build a full **cyber range from scratch** using just a laptop.

Presented at **BSides**, this repo helps students, career-switchers and anyone else spin up realistic lab environments to give themselves hands-on cybersecurity training using the equipment they already have. Features GNS3, Proxmox-based virtualisation, and step-by-step guides to go from zero to homelab hacking ASAP!
